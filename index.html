Configurar Google Drive
Configuração única. Após isso, você só precisará clicar em "Conectar".
Obtenha em console.cloud.google.com → APIs & Services → Credentials
Como obter o Client ID (só uma vez):
1. Acesse console.cloud.google.com
2. Crie um projeto, ative Drive API e Sheets API
3. Credentials → Create Credentials → OAuth 2.0 Client ID
4. Tipo: Web application
5. 🚨 Em Authorized JavaScript Origins adicione:
