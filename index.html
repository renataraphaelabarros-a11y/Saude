⚙️ Configuração
Siga os passos para conectar ao Google Calendar
1
Criar projeto no Google Cloud
Acesse console.cloud.google.com → Crie um projeto novo → Ative a Google Calendar API em "APIs e Serviços".
2
Criar credencial OAuth
Em "Credenciais" → "Criar credenciais" → "ID do cliente OAuth 2.0" → Tipo: Aplicativo da Web.

Em "Origens JavaScript autorizadas", adicione a URL do seu GitHub Pages (ex: https://seunome.github.io).
3
Cole seu Client ID abaixo
Copie o Client ID gerado (formato: xxx.apps.googleusercontent.com)
4
Cole sua API Key da Anthropic (IA)
Acesse console.anthropic.com → API Keys → Crie uma chave.
